Neucruit
============

REQUIREMENTS
------------
- PHP 8.0
- MySQL 8.0
- Composer 2.0
- NodeJS 14

Installation
------------
1. Import the provided SQL file
2. Copy the ".env.example" to ".env" and update the local configuration accordingly
3. Run the following commands:
composer install
npm install --save-dev
php artisan migrate
npm run dev
4. Start coding :)
